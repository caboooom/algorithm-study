1. 매주 월요일마다 3개의 문제를 선정합니다.
2. 같은 주 일요일 오전까지 정해진 문제를 풀고, 소스코드를 업로드합니다.

4. 코드 리뷰는 자유롭게 진행하며, 서로의 코드를 적극적으로 피드백합니다.
